📋 External ID vs. Record ID
Record ID: generated by Salesforce, 15 or 18 characters. The 18-char version is case-insensitive and recommended. Never changes.
External ID: field marked as an external identifier (Text/Number/Email). Used in Upsert to avoid duplicates. Max 25 per object.
🚫 Duplicate Management — Matching & Duplicate Rules
💡Matching Rules define how to identify duplicates (by Email, Name, Phone...). Duplicate Rules decide the action: Allow (with alert), Block, or Report. Available for Leads, Contacts, Accounts, and custom objects.
🔍 Matching Rule: Standard Lead Rule
Field: Email — Exact match
Field: First + Last Name — Fuzzy match
Field: Company — Fuzzy match

Action (on create): Allow + Alert
Action (on edit): Report only
